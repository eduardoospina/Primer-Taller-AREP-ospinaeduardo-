Conversor de temperatura API, conversor de Fahrenheit a Celsius y viceversa.

En esta pagina, se encontrara la estructura del API para la conversion de temperaturas. los posibles errores y ejemplo sde codigo para el mejor entendimiento del funcionamineto de la calculadora de las temperaturas. Manejando unicamente de Celsius y Fahrenheit. Las funcionalidades de esta herramienta se encuentran dentro de las medidas de temperatura mas comunes usadas en el mundo. Por lo que sus aplicaciones son faciles de encontrar.

API Key

El API tiene un identificador unico de modo como un API Key que pasa dentro del API como un parametro de acceso dentro de un url. Este parametro funciona como un identificador unico con las conversiones de temperatura en el API. que para este caso son 2 dependiendo de que conversion se desee hacer.

URL base

https://calc-api-be.herokuapp.com/Calculator/

Para convertir de Celsius a Fahrenheit.

https://calc-api-be.herokuapp.com/Calculator/celsius/32

Para convertir de Fahrenheit a Celsius.

https://calc-api-be.herokuapp.com/Calculator/fahrenheit/320

Todos los resultados se devuelven como JSON. regresando el valor convertido de una temperatura a la otra.

API Response

los datos retornan en un formato standar JSON el cual se logra reparse en cualquier lenguaje de programacion. No tiene en cuenta si se pasa de fahrenheit a celsius o vice versa. la respuesta se encontrara en el formato a continuacion.

Respuesta ejemplo:

{"fahrenheit":320.0,"celsius":160.0}

Al usar el formato le permite la facilidad de hacer uso de esta API por parte de HTTP.
